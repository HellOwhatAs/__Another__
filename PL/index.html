确定性有限状态自动机（DFA，Deterministic Finite Automata）

  • 每一条状态转移规则上的符号都是 ascii 码字符。
  • 从任何一个状态出发，每个字符至多对应一条状态转移规则。

非确定性有限状态自动机（NFA，Deterministic Finite Automata）

  • 每一条状态转移规则上的符号要么是 ascii 码字符，要么是 ϵ。
  • 从一个状态出发，每个符号可能对应多条状态转移规则。
  • 通过 ϵ 转移规则时，不消耗字符串中的字符。

不断的将产生式左边的符号替换成为产生式右边符号的过程称为派生（derivation）。

一个派生中，如果每次都展开最左侧的非终结符，那么这个派生就称为一个最左派生（left-most derivation）；
一个派生中，如果每次都展开最右侧的非终结符，那么这个派生就称为一个最右派生（right-most derivation）；
同一棵解析树能够唯一确定一种最左派生，同一棵解析树能够唯一确定一种最右派生；
如果一串标记串没有歧义，那么只有唯一的最左派生可以生成这一标记串，也只有唯一的最右派生可以生成这一标记串；

规约是派生反向过程

Follow 集合 X 是 Follow( Y ) 的元素当且仅当 ... Y X ... 是可能被完全规约的。

  • 对任意产生式 U -> ... Y Z ... ，First( Z ) 是 Follow( Y ) 的子集。
  • 对任意产生式 Z -> ... Y ，Follow( Z ) 是 Follow( Y ) 的子集。

First 集合 X 是 First( Y ) 的元素当且仅当 X ... 是可能被规约为 Y 的。

  • 对任意终结符 X ， X 都是 First( X ) 的元素。
  • 对任意产生式 Y -> Z ... ，First( Z ) 的元素都是 First( Y ) 的元素。

  • 自反性：对于任意 a ∈ A， a ≤A\le_A≤A​ a
  • 传递性：对于任意 a, b, c ∈ A，如果 a ≤A\le_A≤A​ b、b ≤A\le_A≤A​ c，那么 a ≤A\le_A≤A​ c
  • 反对称性：对于任意 a, b ∈ A，如果 a ≤A\le_A≤A​ b、b ≤A\le_A≤A​ a，那么 a = b

  • Simplify：在 interference graph 中删除度数小于等于 K - 1 的节点，删除的节点用栈记录（如果有move的话，只删除与move无关的节点）
  • Spill：如果无法 Simplify，就任意删除一个节点
  • Select：按照删除节点的倒序为所有变量分配寄存器，Simplify 删除的节点能够保证被分配到与 interference graph 上相邻节点不冲突的寄存器，Spill 变量可能可以分配到寄存器（假 Spill），也可能无法分配到寄存器，此时应当存储在变量中（改写代码后 Start Over）

    #2 = #3 + 1

  • Coalesce：（保守合并）
    如果 x 与 y 两个变量的所有邻居中度数 ≥ K 的数量 ≤ K - 1，那么可以合并 x 与 y
    如果 x 的每个邻居要么也是 y 的邻居，要么度数 ≤ K - 1，那么可以合并 x 与 y
  • Freeze：如果没有可行的 Simplify 和 Coalesce 操作可以继续进行，那么就选择一条 move 指令，放弃对它进行合并
